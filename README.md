# VectorDB - A Python Vector Database

A high-performance vector database implementation for similarity search and retrieval-augmented generation (RAG) applications.

## 🎯 Project Goals

- **Learn**: Understand vector database internals and search algorithms
- **Build**: Create a production-ready vector search engine
- **Scale**: Support from prototypes to production workloads

## 🚀 Features

- [x] In-memory vector storage
- [x] Cosine similarity search
- [ ] FAISS integration for performance
- [ ] Persistent storage
- [ ] Text embedding integration
- [ ] RESTful API
